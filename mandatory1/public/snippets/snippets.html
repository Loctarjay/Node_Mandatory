JSON = JavaScript Object Notation.
It is the way of easier handling information within JS by pairing it all within key/value, even if the value is set to have other key/value pairs.
Express is a framework for easier managing your Web Application as if it was build with the MVC structure.
You can implement a real MVC structure by using a real database but Express is helping you organise and getting ready for that.
Express also helps you save an insane amount of redundant code.
GET - is used when you simply want to open a different page.
Here you're not sending any information to the new page, just opening it
POST - is used when you're sending data to a different page.
Here you're both opening the new page and also sending some data over to the new page, so it can be
handled for some specific purpose.
Cascating Style Sheets is a way to gather the styling same group of objects.
This way you dont need to to every single button, paragraph or other tags inline and remembering which styling you used where, all you need is to remember which class or id the object belongs to.
