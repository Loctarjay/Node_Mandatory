Explanation Picture
Function Declaration is just a standard view of how
to iniate a function, this is done without any complicated
coding involved for the declaration itself
Anonymous Function is when you're not giving the specific
function a direct name to be called on, but only giving
a variable reference to is.
Callback Functions is when you're passing a function as a
parameter to do extra functionalities within the same type
of user action
